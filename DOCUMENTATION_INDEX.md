## 💡 KEY CONCEPTS

### Form Submission

The form submits via standard POST (not AJAX). Hidden fields carry page/search parameters to controller. Controller uses parameters to redirect back to same page with search filter.

### Parameter Preservation

Page and search extracted from URL via URLSearchParams, stored in hidden form fields, posted to controller, used in redirect URL.

### Checkbox State

Value (0 or 1) stored in database. Loaded via data-isrevisi attribute on button. Checked/unchecked state set by JavaScript. Posted in form submission.

### Console Logging

Form submit handler logs extraction and setting of parameters. Server logs track received values and redirect URL. Together they show full flow.
